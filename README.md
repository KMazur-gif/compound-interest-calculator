# Compound Interest Calculator

Console program in Python to calculate compound interest using:
\[
A = P \cdot (1 + r/100)^t
\]

## What it does
- Asks the user for:
  - **principle (P)** — initial amount
  - **interest rate (r)** — yearly interest rate in %
  - **time (t)** — time in years
- Validates that each value is greater than 0
- Prints the final balance formatted to 2 decimals

## Run
```bash
python main.py
